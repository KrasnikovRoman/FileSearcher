Проект: FileSearcher. Автор: Красников Роман Михайлович 

Реализовано:
	1. Вместо консольного приложения, сделал небольшое оконное приложение, используя стандартную библиотеку Swing.
	2. Базовый поиск файлов по заданному имени в рабочей директории. На выходе выводится информация о файле в новом окне в виде таблицы.
	3. Реализована поддержка регулярных выражений. В строке поиска необходимо ввести регулярное выражение.
	4. При нажатой кнопке "Additional information" выводятся дополнительные атрибуты файлов: размер, время создания, время последнего изменения.
	5. При вводе в строку поиска символа "*" выводится информация о всех файлах и папках, находящихся в рабочей  директории.
